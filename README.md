## Chatbot

Simple web-based chatbot UI.

- **Open**: Double-click `chatbot.html` in a browser.
- **Style**: Edit `style.css` to change the look.

This folder contains a simple web-based chatbot built with HTML, CSS, and vanilla JavaScript.

### Files

- **chatbot.html**: Main page that displays the chatbot UI.
- **style.css**: Styles for the chatbot layout and appearance.

### How to run

- **Option 1 (recommended)**: Double-click `chatbot.html` to open it directly in your browser.
- **Option 2**: Open this folder in a code editor and use a simple local server (for example, with an extension like “Live Server”) to serve `chatbot.html`.

### Customization

- **Appearance**: Edit `style.css` to change colors, fonts, spacing, or layout.
- **Behavior / responses**: Open `chatbot.html` and adjust the embedded JavaScript logic (if present) to change how the chatbot responds to messages.

### Requirements

- A modern web browser (Chrome, Edge, Firefox, etc.).
